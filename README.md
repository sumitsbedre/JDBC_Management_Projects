# JDBC_Management_Projects
Basic JDBC management projects

Employee & Student Management System | Java Backend Project

## Technologies: 
- Java 
- JDBC
- PostgreSQL
- Maven

## What you can get here: 
-> Built a console-based backend system to manage employee and student records using Core Java and JDBC.
-> Implemented full CRUD operations (Create, Read, Update, Delete) with PostgreSQL database integration.
-> Designed a layered architecture with Controller, DAO, Entity, and Connection modules for modular and maintainable code.
-> Implemented database connectivity and query handling using PreparedStatement and JDBC APIs.
-> Developed two independent modules: Employee Management and Student Management, each with dedicated database schemas.

## What are Provided
### Project files consist of:
-  Connection Layer files
-  Dao Layer (Data-Access-Object)
-  Enitity Layer
-  Controller Layer
